About

我叫 zenus，全栈 AI 应用开发者。三个月前主动辞职，为追求「相对自由」——不是暴富，是有选择权。

程序员，也是修行者。修行让我能觉察情绪而不被裹挟，做决定之前有一个停顿。

目前在做的两件事：桌面宠物应用（做一个尽可能真实的猫）和内容探索（用 AI 延伸自己，替我产出内容）。目标是持续复利，让系统跑，我不在流程里。

这是我的真实记录，不给答案，只给过程。

220 · 浅谈 Agent Memory

Memory 不是「更聪明的检索」，而是系统在时间里维持连续性。CoALA 框架将记忆分为工作/情景/语义/程序四类。RAG 是图书馆，Memory 是对你的理解。核心难点：写入质量、整理（让旧版本退场）、读取（为当下重建关于过去的工作假说）。Raw 与 Derived 的平衡——没有证据层会漂移，没有派生层会钝化。

219 · 深度解析：Claude Code 源码

源码适合当约束样本、病理切片、反证材料三类同时阅读。系统真正维护四条不变量：轨迹拓扑不破、缓存前缀不漂、能力面不一次性敞开、连续性不只寄托在 transcript。核心模块：query.ts（编排中心）、attachments.ts（第二调度器）、tool mediation、continuous surface 外化。源码过滤器：principle > workaround > legacy bridge > debt。
