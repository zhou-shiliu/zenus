# Identity System

这份文档用于统一 `zenus` 个人网站与相关社交账号的身份表达，避免名字体系过满、过重或使用混乱。

## 名字体系

- **主名 / 现实识别层**：小兵 / zenus
- **字 / 人格表达层**：明则
- **号 / 精神意象层**：观火

## 使用总原则

1. **先让别人认出你是谁，再让别人理解你。**
2. 能用 **小兵 / zenus** 的地方，不先上 **明则**。
3. 能用 **明则** 的地方，不先上 **观火**。
4. **观火** 低频使用，适合栏目名、专题名、About 页面、长文自序，不适合做所有平台的常驻后缀。
5. 网站首页和大多数社交简介，优先突出“你在做什么”，而不是完整展开“名、字、号”。

## 最终使用顺序

长期对外表达时，按这个顺序出现：

**小兵 / zenus → 明则 → 观火**

也就是：
- 先让别人知道你是谁
- 再让别人感受到你的人格气质
- 最后才让真正愿意深入的人看到你的精神系统

## 一句话总结

**主名负责识别，明则负责人格，观火负责深层气质。**
